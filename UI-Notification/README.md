# UI / Notification 扩展

这个目录收录与界面提示、通知、反馈展示相关的 Pi 扩展。

## 已收录
- [pi-jarvis](./pi-jarvis/) - `/jarvis` 侧边对话 overlay，为主会话提供独立辅助线程、权限开关与回传主线程能力
- [pi-thinking-steps](./pi-thinking-steps/) - thinking 显示增强扩展，提供 collapsed / summary / expanded 三种终端推理视图
- [pi-draw](./pi-draw/) - 通过 `/draw` 或快捷键打开 tldraw 画布，将绘图导出为 PNG 并附加到当前 prompt
